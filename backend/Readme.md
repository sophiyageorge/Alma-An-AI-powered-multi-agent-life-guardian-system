# Wellness Guidance and Lifestyle Assistance

## Project Overview
This is an AI-driven wellness and lifestyle guidance system with multiple agents: Nutrition, Exercise, Health Data, Grocery, Doctor Booking, Mental Health, Emergency, and Compliance. Supports Family and Bachelor modes.

## Tech Stack
- Backend: FastAPI
- Frontend: Django (to be integrated)
- Database: PostgreSQL/MySQL
- Messaging/Orchestration: Celery + RabbitMQ
- AI Agents: LangGraph, TensorFlow/PyTorch
- Deployment: Docker
